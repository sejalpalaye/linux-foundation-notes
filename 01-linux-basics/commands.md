# Linux Basics – Command Line Foundations

## 📌 Introduction

This section documents my foundational learning of Linux command-line operations.  
The goal is to build strong system-level understanding by practicing essential commands used in daily Linux administration.

Through hands-on practice, I am learning:
- File system navigation
- Directory and file management
- Basic command usage
- Understanding Linux terminal behavior

These fundamentals form the base for system administration, cybersecurity, and cloud environments.


## 🖥️ Basic Commands
These commands are essential for navigating and managing the Linux file system through the terminal.

### 1️⃣ pwd
**Description:** Displays the current working directory.

Bash:
[sejalpalaye@localhost ~]$ pwd  
/home/sejalpalaye
